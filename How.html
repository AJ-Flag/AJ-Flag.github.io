Principles of page design

Contents to a webpage should be easy to digest. If needing to navigate a site it should be easy for a user to do so. Also, keeping in line where certain elements should be, like having the "About Us" link at the bottom of every page.

Home Page

A home page shouldn't be littered with links and tons of images. It should be concise with it's point and navigation.

Navigation

This step is essential in website design; if it becomes overly confusing, users are unlikely to continue using the interface. If users can sense progress from one step to the next and can feel that they're making their way to their final goal, then they'll remain on a site.

Common Elements

This part requires consistency across the entire page. Having the site's logo appear in the top right/left corner to indicate possesion of site. Typically have the header be the place to find navigation tools like links or search bars. An additional element is the footer having items like "Credits" or "Terms & Conditions".

Context

Two of the main things a site should be able to complete is relevant content and headings. Relevant content should be special for each site. Headings should be used to grab attention and show what the upcoming content is about like how the word "Context" is big and bold.

Overview

How to design a website? Designing a website means to have a purpose and the funcionality be comprehendable. All while making everything accessible.

To design a good website means to follow the guidelines off the principles of page design. Components such as Home Page, Navigation, Common Elements, and Context need to be considered for every page on every site. There's numerous tools to make internet usage accessible for everyone, one of the most commercialized tool is the screen reader. A page should make the employment of screen readers smooth so the tool doesn't skip a beat with translating text to speech.

Screen Readers

This tool translates on screen text to an audible speech pattern played through one's speakers or headphones. People with visual impairments benefit from screen readers as they're able to understand a page's content. Additionally, people with age-related impairments, health condtions, and/or anyone who perfers an audio median than a visual one will usually perfer screen readers.
